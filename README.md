# PyGraphicsMap

[![N|Solid](https://cldup.com/dTxpPi9lDf.thumb.png)](https://nodesource.com/products/nsolid)

[![Build Status](https://travis-ci.org/joemccann/dillinger.svg?branch=master)](https://travis-ci.org/joemccann/dillinger)

PyGraphicsMap library can be used for software development in Python programming language, where work with dynamic objects on the cartographic basis is required and no need to use special GIS like ArchGIS or QGIS

# Install

Adding to pip in process

# Example

If you want to use example you need to uncomment lines 36-42 in [main.py](https://github.com/Fynduk/PyGraphicsMap/blob/master/ExampleApp/main.py)
```python
    # Example how to connect planes to project
    self.__thread_for_planes = QThread()
    self.flight = PlaneManager(scene)
    self.__thread_for_planes.start()
    self.flight.moveToThread(self.__thread_for_planes)
    self.threadPlaneManager.connect(self.flight.createPlanes)
    self.__thread_for_planes.finished.connect(self.flight.deleteLater)
    self.threadPlaneManager.emit()
```
